# Security Policy

## Secrets

Do not commit API keys, Feishu credentials, cookies, browser profiles, OpenClaw
state, logs from customer machines, or generated PDF reports containing customer
data.

The community edition does not require DeepSeek, Feishu, OpenClaw, or Buyin
credentials.

## Reporting

Please open a private GitHub security advisory or contact the repository owner
if you find a secret exposure, command-injection risk, or unsafe browser
automation pattern.
