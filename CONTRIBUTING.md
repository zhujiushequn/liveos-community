# Contributing

Thanks for improving LiveOS Community.

Please keep contributions inside the community boundary:

- Add interfaces, examples, tests, docs, and local demo behavior.
- Do not copy commercial Feishu automation, Buyin/Juliang Baiying automation,
  private OpenClaw routing, customer logs, prompts, or credentials.
- Do not commit generated media files, PDFs, browser profiles, cookies, or API
  keys.

Before opening a pull request:

```bash
python -m unittest discover -s tests -v
```
