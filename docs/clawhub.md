# ClawHub Publishing

The ClawHub artifact for this repository is a skill folder:

```text
skills/liveos-community/SKILL.md
```

Publish with:

```bash
npm i -g clawhub
clawhub login
clawhub skill publish skills/liveos-community --dry-run
clawhub skill publish skills/liveos-community
```

ClawHub skills are published under MIT-0 by the registry. The GitHub source
repository remains MIT licensed.
