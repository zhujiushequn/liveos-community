# Open-Core Boundary

This repository is the public community layer.

## Included

- Public event model
- Public-screen message classification
- Sample data
- Local Markdown report generation
- Receiver, collector, and reporter interfaces
- A ClawHub skill for agents that need to understand the community repo

## Excluded

- Feishu robot creation, publishing, permission grants, and WebSocket receiver
- OpenClaw `anchor` production routing and `JOB_STARTED` proof writer
- DeepSeek production configuration and prompt libraries
- Juliang Baiying / Buyin login, account switching, same-source monitor, replay
  monitor, and hidden KPI extraction
- Windows one-click offline runtime pack, repair kits, remote support scripts,
  and customer-machine diagnostics
- Central knowledge base and complete commercial report logic

## Why

The open repository should prove the technical shape without exposing customer
delivery mechanics or proprietary operating knowledge.
